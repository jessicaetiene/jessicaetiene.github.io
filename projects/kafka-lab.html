O que estou estudando neste laboratório

Este projeto funciona como um laboratório prático para registrar meus estudos sobre Apache Kafka e consolidar fundamentos de mensageria orientada a eventos.

  • Criação de tópicos.
  • Criação e evolução de schemas Avro.
  • Envio de mensagens para tópicos (producers).
  • Consumo de mensagens (consumers).
  • Uso de Docker para subir um cluster Kafka local.
